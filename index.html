Perfect Responsive Columns.

Perfect Responsive Columns is a 10 columns based responsive grid system.

Classes

.one_tenth = 10
.one_ninth = 9
.one_eighth = 8
.one_seventh = 7
.one_sixth = 6
.one_fifth = 5
.one_fourth = 4 or 1/4
.one_third = 3 or 1/3
.one_half = 2
.two_thirds = 2/3
.full_width = 1

$ cd your_repo_root/Perfect-Responsive-Columns
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
